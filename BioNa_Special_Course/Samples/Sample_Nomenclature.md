# Sample Nomenclature & Pyrolysis Matrix

## Naming Convention

```
MMB1 - A10 - 3Crm
│       │       └─ Ramp rate suffix (only for 1200 °C samples): xCrm = x °C/min
│       │          Format avoids "/" to be path-safe (e.g. 1Crm, 3Crm, 10Crm)
│       └────────── Temperature code: digit × 100 = pyrolysis temp (°C)
└────────────────── Material series: MMB1 = BioNa standard biochar
                    (MMB2 exists but is not used in this study)
```

> The "A" series letter meaning is currently unknown — preserve as-is.  
> Ramp rate suffix only added for 1200 °C samples (the other temperatures use 3 °C/min only).

## Notes

- Lower pyrolysis temperatures (600, 800 °C) yield less well-carbonized hard carbon — expect lower capacity and more disordered structure (larger d-spacing, smaller La/Lc).
- 1000 °C (A10) is the reference/standard temperature.
